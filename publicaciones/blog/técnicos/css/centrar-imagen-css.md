Para conseguir que una imagen quede centrada en nuestra página web necesitaremos hacer uso de tres propiedades. La primera será definir `display` con el valor `block`.

```css
.img-centrada {
	display: block;
	margin-left: auto;
	margin-right: auto;
}
```
